# Site Vitrine - Plataforma de Streaming

Landing page simples para apresentação da plataforma de streaming e captação de contatos via WhatsApp.

## Objetivo

Mostrar a proposta de valor do serviço de forma direta, com:

- destaque principal (hero)
- benefícios principais
- resumo de planos
- chamada para ação

## Estrutura

- `src/pug/index.pug`: estrutura da página
- `src/scss/`: estilos da landing page
- `dist/`: versão compilada para publicação

## Como rodar localmente

1. `npm install`
2. `npm run build`
3. abrir `dist/index.html`

## Observação

Este projeto foi adaptado com base em um template Bootstrap, mas está customizado para uso como vitrine da Plataforma de Streaming.
